Flexbox for positioning elements. 2 components: Flex CONTAINERS & Flex ITEMS. To designate an element as a flex container: display: flex; OR display: inline-flex; ONCE an item is a flex container, several properties to specify how its children behave: 1. justify-items 2. align-items 3. flex-grow 4. flex-shrink 5. flex-basis 6. flex 7. flex-wrap 8. align-content 9. flex-direction 10. flex-flow
